# 🌐 Meus Sites Favoritos


---

## 🛠️ Tecnologias Utilizadas

- **HTML5**  
- Editor de código (VS Code)  
- Navegador para visualização (Edge,)

---

## 📄 Estrutura do Projeto

O projeto contém:
- Um arquivo principal: `index.html`
- Uma pasta com as imagens dos sites (`youtube.png`, `netflix.png`, `honkaiwiki.png`, `spotify.png`, `phasmophobia.png`)

---

## 🧩 Funcionalidades

- Mostra 5 sites favoritos com:
  - Nome do site
  - Imagem representativa
  - Link direto para o site
